# Utilities

Small, focused tools used across Obsmind projects. Each tool lives in its own folder with a dedicated README and setup steps.

## Available tools

### compress-pdf
Compress PDFs in place using Ghostscript with an optional Finder Quick Action for one-click compression.

- Folder: `compress-pdf/`
- Use cases: shrink manuals before sharing, speed up uploads, and keep archives lightweight.
- Works on macOS with Homebrew Ghostscript.
